Let's create a simple mod that adds a custom message when you join a world.

## Prerequisites

- BepisLoader installed and working
- Visual Studio 2022 or VS Code with C# extension
- .NET 9 SDK installed
- Basic C# knowledge

## Project Setup

### 1. Create a New Project

Create a new Class Library project:

```bash
dotnet new classlib -n MyFirstResoniteMod -f net9.0
cd MyFirstResoniteMod
```

### 2. Add Dependencies

Edit your `.csproj` file:

```xml
<Project Sdk="Microsoft.NET.Sdk">
  <PropertyGroup>
    <TargetFramework>net9.0</TargetFramework>
    <LangVersion>latest</LangVersion>
  </PropertyGroup>

  <ItemGroup>
    <PackageReference Include="BepInEx.Core" Version="6.0.0-be.*" />
    <PackageReference Include="HarmonyX" Version="2.10.2" />
  </ItemGroup>

  <ItemGroup>
    <Reference Include="FrooxEngine">
      <HintPath>$(ResonitePath)\Resonite_Data\Managed\FrooxEngine.dll</HintPath>
    </Reference>
    <Reference Include="Elements.Core">
      <HintPath>$(ResonitePath)\Resonite_Data\Managed\Elements.Core.dll</HintPath>
    </Reference>
  </ItemGroup>
</Project>
```

Set the `ResonitePath` environment variable to your Resonite installation folder.

## Writing the Mod

### 3. Create the Plugin Class

Create `MyFirstMod.cs`:

```csharp
using BepInEx;
using BepInEx.Logging;
using HarmonyLib;
using FrooxEngine;
using Elements.Core;

namespace MyFirstResoniteMod
{
    [BepInPlugin(PLUGIN_GUID, PLUGIN_NAME, PLUGIN_VERSION)]
    public class MyFirstMod : BaseUnityPlugin
    {
        public const string PLUGIN_GUID = "com.example.myfirstmod";
        public const string PLUGIN_NAME = "My First Resonite Mod";
        public const string PLUGIN_VERSION = "1.0.0";

        internal static ManualLogSource ModLog;

        private void Awake()
        {
            ModLog = Logger;
            ModLog.LogInfo($"{PLUGIN_NAME} v{PLUGIN_VERSION} loading...");

            // Apply Harmony patches
            var harmony = new Harmony(PLUGIN_GUID);
            harmony.PatchAll();

            ModLog.LogInfo($"{PLUGIN_NAME} loaded successfully!");
        }
    }
}
```

### 4. Add a Harmony Patch

Create `WorldJoinPatch.cs`:

```csharp
using HarmonyLib;
using FrooxEngine;
using Elements.Core;

namespace MyFirstResoniteMod
{
    [HarmonyPatch(typeof(World), "Join")]
    public static class WorldJoinPatch
    {
        static void Postfix(World __instance, User user)
        {
            // Check if it's the local user joining
            if (user.IsLocalUser)
            {
                MyFirstMod.ModLog.LogInfo($"Welcome to {__instance.Name}!");
                
                // Send a notification in-game
                NotificationMessage.SpawnTextMessage(
                    "Welcome!",
                    $"My First Mod says hello from {__instance.Name}!",
                    colorX.Green
                );
            }
        }
    }
}
```

## Building and Installing

### 5. Build the Mod

```bash
dotnet build -c Release
```

### 6. Install the Mod

Copy the built DLL from `bin/Release/net9.0/MyFirstResoniteMod.dll` to your Resonite `BepInEx/plugins/` folder.

### 7. Test the Mod

1. Launch Resonite
2. Join any world
3. You should see a green notification message
4. Check `BepInEx/LogOutput.log` for your log messages

## Adding Configuration

### 8. Make It Configurable

Update `MyFirstMod.cs` to add configuration:

```csharp
using BepInEx.Configuration;

public class MyFirstMod : BaseUnityPlugin
{
    // ... previous code ...
    
    private ConfigEntry<string> WelcomeMessage;
    private ConfigEntry<bool> ShowNotification;

    private void Awake()
    {
        // Configure settings
        WelcomeMessage = Config.Bind(
            "General",
            "WelcomeMessage", 
            "Welcome to Resonite!",
            "The message to display when joining a world"
        );
        
        ShowNotification = Config.Bind(
            "General",
            "ShowNotification",
            true,
            "Whether to show the in-game notification"
        );

        // ... rest of Awake method ...
    }
    
    internal static MyFirstMod Instance { get; private set; }
    
    private void Awake()
    {
        Instance = this;
        // ... rest of initialization ...
    }
}
```

Update `WorldJoinPatch.cs`:

```csharp
static void Postfix(World __instance, User user)
{
    if (user.IsLocalUser && MyFirstMod.Instance.ShowNotification.Value)
    {
        NotificationMessage.SpawnTextMessage(
            "Welcome!",
            MyFirstMod.Instance.WelcomeMessage.Value,
            colorX.Green
        );
    }
}
```

## Debugging Tips

### Enable Debug Logging

```csharp
#if DEBUG
    ModLog.LogDebug("This only shows in debug builds");
#endif
```

### Use Try-Catch Blocks

```csharp
try
{
    // Your mod code
}
catch (Exception ex)
{
    ModLog.LogError($"Error in mod: {ex}");
}
```

### Performance Considerations

- Use Harmony patches sparingly
- Cache references when possible
- Avoid heavy operations in frequently-called methods

## Common Patterns

### Accessing the Local User

```csharp
var localUser = __instance.LocalUser;
if (localUser != null)
{
    // Do something with local user
}
```

### Working with Slots

```csharp
var rootSlot = __instance.RootSlot;
var newSlot = rootSlot.AddSlot("MyModSlot");
```

### Using Coroutines

```csharp
StartCoroutine(MyCoroutine());

private IEnumerator MyCoroutine()
{
    yield return new WaitForSeconds(1f);
    // Do something after delay
}
```

## Publishing Your Mod

### Create a README

Include:
- What your mod does
- Installation instructions
- Configuration options
- Known issues
- Credits

### Upload to Thunderstore

1. Create a `manifest.json`:
```json
{
    "name": "MyFirstResoniteMod",
    "version_number": "1.0.0",
    "website_url": "https://github.com/yourusername/MyFirstResoniteMod",
    "description": "A simple welcome message mod",
    "dependencies": ["BepInExPack_Resonite-1.0.0"]
}
```

2. Package your mod with icon.png (256x256) and README.md
3. Upload to [Thunderstore](https://thunderstore.io/c/resonite/)

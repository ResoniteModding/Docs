## Core Development Tools

### Mod Loaders

#### BepisLoader

**Purpose**: Primary mod loading framework for Resonite

- **Download**: [Latest Release](https://github.com/art0007i/BepisLoader/releases/latest/download/BepisLoader-full.zip)
- **Features**:
  - Full BepInEx integration
  - Windows and Linux support
  - Automatic dependency management
  - Easy installation process

**Installation**:

1. Extract `BepisLoader-full.zip` over Resonite folder
2. (Linux) Modify `LinuxBootstrap.sh` to launch `BepisLoader.dll`

## Decompilation & Analysis Tools

### ILSpy (Free)

**Purpose**: Decompile .NET assemblies to C#

- **Download**: [GitHub Releases](https://github.com/icsharpcode/ILSpy/releases)
- **Use Cases**:
  - Examine Resonite's code structure
  - Understand component implementations
  - Find method signatures for patching
  - Explore third-party mods

**Key Features**:

- Multiple output formats (C#, IL, VB.NET)
- Search functionality
- Assembly comparison
- Plugin architecture

### dotPeek (Free)

**Purpose**: JetBrains' .NET decompiler

- **Download**: [JetBrains Website](https://www.jetbrains.com/decompiler/)
- **Advantages**:
  - Excellent code navigation
  - Integration with ReSharper
  - Assembly explorer
  - Export to project format

## Debugging Tools

### Built-in .NET Debugger

**Setup for Resonite**:

1. Build mod in Debug configuration
2. Copy PDB files to plugins folder
3. Launch Resonite
4. Attach debugger to `Resonite.exe`

**Debugging Tips**:

- Use conditional breakpoints for performance
- Watch window for complex objects
- Call stack analysis for patch issues

### Custom Debug Tools

#### Performance Profiler

```csharp
public class SimpleProfiler
{
    private static Dictionary<string, (long ticks, int calls)> timings = new();

    public static void Profile(string name, Action action)
    {
        var sw = Stopwatch.StartNew();
        action();
        sw.Stop();

        if (timings.TryGetValue(name, out var existing))
            timings[name] = (existing.ticks + sw.ElapsedTicks, existing.calls + 1);
        else
            timings[name] = (sw.ElapsedTicks, 1);
    }

    public static void PrintReport()
    {
        foreach (var (name, (ticks, calls)) in timings.OrderByDescending(x => x.Value.ticks))
        {
            var ms = (ticks * 1000.0) / (Stopwatch.Frequency * calls);
            MyMod.Logger.LogInfo($"{name}: {ms:F2}ms avg ({calls} calls)");
        }
    }
}
```

#### Memory Monitor

```csharp
public static void LogMemoryUsage(string context)
{
    var beforeCollection = GC.GetTotalMemory(false);
    GC.Collect();
    GC.WaitForPendingFinalizers();
    GC.Collect();
    var afterCollection = GC.GetTotalMemory(false);

    MyMod.Logger.LogInfo($"{context} - Memory: {beforeCollection / 1024 / 1024}MB -> {afterCollection / 1024 / 1024}MB");
}
```

### Automated Deployment

#### PowerShell Deploy Script

```powershell
param(
    [string]$Configuration = "Debug",
    [string]$ResonitePath = $env:RESONITE_PATH
)

$ModName = "MyMod"
$ProjectPath = ".\$ModName.csproj"
$PluginsPath = "$ResonitePath\BepInEx\plugins"

Write-Host "Building $ModName ($Configuration)..."
dotnet build $ProjectPath -c $Configuration

if ($LASTEXITCODE -eq 0) {
    Write-Host "Copying to Resonite..."
    Copy-Item ".\bin\$Configuration\net9.0\$ModName.dll" $PluginsPath
    Copy-Item ".\bin\$Configuration\net9.0\$ModName.pdb" $PluginsPath
    Write-Host "Deployment complete!"
} else {
    Write-Error "Build failed!"
}
```

#### Bash Deploy Script

```bash
#!/bin/bash
MOD_NAME="MyMod"
CONFIG="Debug"
RESONITE_PATH="${RESONITE_PATH:-$HOME/.local/share/Steam/steamapps/common/Resonite}"

echo "Building $MOD_NAME ($CONFIG)..."
dotnet build "$MOD_NAME.csproj" -c "$CONFIG"

if [ $? -eq 0 ]; then
    echo "Copying to Resonite..."
    cp "./bin/$CONFIG/net9.0/$MOD_NAME.dll" "$RESONITE_PATH/BepInEx/plugins/"
    cp "./bin/$CONFIG/net9.0/$MOD_NAME.pdb" "$RESONITE_PATH/BepInEx/plugins/"
    echo "Deployment complete!"
else
    echo "Build failed!" >&2
    exit 1
fi
```

## Monitoring and Analytics

### Log Analysis Tools

#### LogTail Script

```powershell
# PowerShell log monitor
param([string]$LogPath = "BepInEx\LogOutput.log")

Get-Content $LogPath -Wait -Tail 10 |
    Where-Object { $_ -match "MyMod" } |
    ForEach-Object {
        $timestamp = Get-Date -Format "HH:mm:ss"
        Write-Host "[$timestamp] $_"
    }
```

#### Performance Counter

```csharp
public class PerformanceCounter
{
    private readonly Dictionary<string, List<double>> metrics = new();

    public void Record(string metric, double value)
    {
        if (!metrics.ContainsKey(metric))
            metrics[metric] = new List<double>();
        metrics[metric].Add(value);
    }

    public void PrintSummary()
    {
        foreach (var (metric, values) in metrics)
        {
            var avg = values.Average();
            var min = values.Min();
            var max = values.Max();
            MyMod.Logger.LogInfo($"{metric}: avg={avg:F2}, min={min:F2}, max={max:F2}");
        }
    }
}
```

### Resonite-Specific References
These DLLs are found in your Resonite installation (`Resonite_Data/Managed/`):
- `FrooxEngine.dll` - Core game engine
- `Elements.Core.dll` - Utilities and data structures  
- `Elements.Assets.dll` - Asset management
- `CloudX.Shared.dll` - Cloud functionality

## Getting Help

### When You're Stuck
1. **Check existing documentation** - Start with this wiki
2. **Search the Discord** - Someone may have asked before
3. **Ask in #mod-development** - The community is helpful
4. **Share your code** - Use pastebin or GitHub gists
5. **Provide error logs** - Include full stack traces

### Best Practices for Questions
- Be specific about your problem
- Include relevant code snippets
- Share error messages verbatim
- Mention what you've already tried
- Describe your expected vs actual behavior

### Contributing Back
- Share working code snippets
- Document solutions you find
- Help other beginners
- Report bugs in tools
- Submit improvements to this wiki

## Quick Reference

### Recommended Launch Arguments
```bash
-SkipIntroTutorial -DoNotAutoLoadHome -LogFile ResoniteLog.txt
```

### File Locations
- **Mods**: `Resonite/BepInEx/plugins/`
- **Config**: `Resonite/BepInEx/config/`
- **Logs**: `Resonite/BepInEx/LogOutput.log`
- **Game Data**: `Resonite/Resonite_Data/Managed/`
